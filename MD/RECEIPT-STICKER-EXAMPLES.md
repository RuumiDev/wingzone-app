## Print Layout for Stickers

When printing stickers, the system displays them in a 2-column grid:

```
+------------------+  +------------------+
|   STICKER 1      |  |   STICKER 2      |
|   (Alice)        |  |   (Bob)          |
+------------------+  +------------------+

+------------------+  +------------------+
|   STICKER 3      |  |   STICKER 4      |
|   (Charlie)      |  |   (Diana)        |
+------------------+  +------------------+
```

Each sticker prints on its own 2"x3" label with page breaks between them.

## QR Code Details

**Receipt QR Codes:**
- Size: 120x120px
- Format: `WINGZONE-ORDER-{orderId}`
- Level: M (medium error correction)
- Border: 2px solid border

**Sticker QR Codes:**
- Size: 60x60px
- Format: `WINGZONE-{lobbyCode}-{userName}-{boxNumber}`
- Level: M (medium error correction)
- Positioned in bottom-right corner

**Scannable Data:**
- Order tracking
- User identification
- Box verification
- Lobby association

## Color Coding (For Screen Preview)

**Receipt:**
- Black text on white background
- Red accents for branding
- Gray for secondary info

**Stickers:**
- Red background for user name section
- Yellow background for drink checkbox
- White for item details
- Black borders and text

**Print Output:**
- Thermal printers render in black & white
- Color coding helps with screen preview
- Grayscale conversion automatic
